# HYP Otomasyon - Kurulum Kılavuzu

## Hızlı Başlangıç

1. **HYP_Otomasyon.exe** dosyasını çift tıklayın
2. İlk çalıştırmada Chrome tarayıcı kontrolü yapılır
3. Chrome yoksa otomatik kurulum önerilir
4. Program açıldıktan sonra HBYS bilgilerinizi girin

---

## Sistem Gereksinimleri

| Gereksinim | Minimum |
|------------|---------|
| İşletim Sistemi | Windows 10/11 |
| RAM | 4 GB |
| Disk Alanı | 200 MB |
| İnternet | Gerekli |
| Tarayıcı | Google Chrome (otomatik kurulur) |

---

## Kurulum Adımları

### Adım 1: Dosyayı Kopyalayın
- `HYP_Otomasyon.exe` dosyasını bilgisayarınıza kopyalayın
- Masaüstüne veya istediğiniz bir klasöre yerleştirin

### Adım 2: Programı Çalıştırın
- Dosyaya çift tıklayın
- Windows güvenlik uyarısı çıkarsa "Yine de çalıştır" seçin

### Adım 3: Chrome Kontrolü
- Program otomatik olarak Chrome tarayıcısını kontrol eder
- Chrome kurulu değilse size sorar:
  - **Evet**: Chrome otomatik indirilip kurulur
  - **Hayır**: Manuel kurmanız gerekir

### Adım 4: HBYS Giriş Bilgileri
- Açılan ekranda HBYS kullanıcı adı ve şifrenizi girin
- "Beni Hatırla" seçeneği ile bilgileriniz güvenli şekilde saklanır

---

## Sık Karşılaşılan Sorunlar

### "Bilinmeyen yayımcı" uyarısı
Windows Defender bu uyarıyı gösterebilir.
- "Ek bilgi" veya "Yine de çalıştır" seçeneğini tıklayın
- Program güvenlidir, dijital imza eklenmemiştir

### Chrome otomatik kurulmuyor
Manuel kurulum için:
1. https://www.google.com/chrome/ adresine gidin
2. Chrome'u indirip kurun
3. HYP Otomasyon'u tekrar başlatın

### Program açılmıyor
1. Bilgisayarı yeniden başlatın
2. Antivirüs programınızı geçici olarak kapatın
3. Programı yönetici olarak çalıştırın (sağ tık > Yönetici olarak çalıştır)

### HBYS'ye bağlanamıyor
1. İnternet bağlantınızı kontrol edin
2. HBYS sisteminin çalıştığından emin olun
3. Kullanıcı adı ve şifrenizi kontrol edin
